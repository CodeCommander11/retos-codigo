# 🎯 Retos de Código y Prácticas de Ciencia de Datos

¡Hola! En este repositorio almaceno mis ejercicios prácticos, retos de código semanales y proyectos pequeños desarrollados durante mi formación en Ciencia de Datos y Modelado Predictivo enfocado en la toma de decisiones.

### 📂 Estructura del Repositorio
*   `/platzi/`: Ejercicios y laboratorios de las rutas de aprendizaje de Platzi.
*   `/retos-semanales/`: Soluciones a problemas de lógica, estadística y algoritmos.
*   `/pruebas/`: Notebooks experimentales de código en Python y consultas SQL.

### 🛠️ Tecnologías Utilizadas
*   **Lenguaje Principal:** Python (Pandas, NumPy, Scikit-Learn).
*   **Entorno de Trabajo:** Google Colab.
*   **Bases de Datos:** SQL.
